## Tabela IMC com JavaScript
Uma Site com tabela de IMC, que contém os dados de:
- Magreza
- Normal
- Sobrepeso
- Obsidade

## Tecnologias usadas
- HTML
- CSS
- JavaScript

## Como Funciona?
1. O usuário põe o valor no input.
2. E clica no botão gerar.
3. O sistema JS verifica se o valor existe e que seja númerico.
4. Logo, passa pelo dados salvos no script, e mostra o resultado do IMC e a tabela IMC
5. Para voltar para página inicial, basta clicar no botão voltar.
